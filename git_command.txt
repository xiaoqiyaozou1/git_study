基本命令
1. git init //初始化本地git 仓库
2. git add <file> 添加文件
3. git status //查看状态
4. git commit //提交
5. git push //推送到仓库
6. git pull //从远程仓库拉取数据
7. git clone //从远程仓库拷贝数据
8. git rm --cached <file> //移除临时仓库的文件
9. git add .  //将所有的文件上传
如何忽略文件和文件夹
1. 创建'.gitignore' 文件
2. 忽略文件 直接写文件的名字
3. 忽略文件夹 写相对于此目录的文件路径 如 ： '/dir'
创建分支等命令
1. git branch 'name' 创建分支名称
2. git checkout 'name' 切换到分支下
3. git merge 'branch name' 在主目录下合并分支目录
